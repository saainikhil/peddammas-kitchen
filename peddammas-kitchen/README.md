# Peddamma's Kitchen Website

This repository contains a responsive, modern website template for Peddamma's Kitchen — a homemade Andhra-style food brand.

Pages:
- index.html (home with slider & featured products)
- products.html (menu searchable)
- product-details.html (single product view)
- about.html
- contact.html
- bulk-orders.html

Structure: css/, js/, assets/, components/

To preview: open `index.html` in a browser (no build required).
